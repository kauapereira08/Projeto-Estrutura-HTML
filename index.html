CSS

Conteúdo: CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o layout de documentos HTML. Ele permite personalizar elementos como fontes, cores, espaçamento, bordas, alinhamento, entre outros. Com CSS, é possível estilizar não apenas o conteúdo HTML, mas também elementos como tabelas, listas, parágrafos e até mesmo linhas e bordas. Como exemplo ficam as estilizações que realizamos nesta página web.

Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .css. JAVA SCRIPT

JavaScript (JS) é uma linguagem de programação utilizada principalmente para adicionar interatividade e dinamismo a páginas web. Com o JS, é possível criar efeitos visuais, validar formulários, manipular elementos da página, criar animações e até mesmo realizar requisições assíncronas para atualizar conteúdo sem recarregar a página.

Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .js.
